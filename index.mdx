<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold dark:text-white text-gray-900 mb-4">Quick start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get your API up and running in minutes</p>
  
  <Steps>
    <Step title="Clone and configure">
      Clone the repository and set up your JWT configuration:
      
      ```bash
      git clone https://github.com/0xfelaback/Social-Media-Activity-Feed.git
      cd Social-Media-Activity-Feed
      dotnet user-secrets set "Jwt:Issuer" "your-issuer"
      dotnet user-secrets set "Jwt:Key" "your-secret-key-here"
      ```
    </Step>
    
    <Step title="Set up the database">
      Run EF Core migrations to create your SQLite database:
      
      ```bash
      dotnet ef database update
      ```
      
      <Note>The API uses SQLite for local development. For production, consider migrating to PostgreSQL.</Note>
    </Step>
    
    <Step title="Start the API">
      Launch the API server:
      
      ```bash
      dotnet run
      ```
      
      Your API will be available at `http://localhost:5000` (or the port specified in your launch settings).
    </Step>
    
    <Step title="Make your first request">
      Register a new user account:
      
      ```bash cURL
      curl -X POST http://localhost:5000/api/register \
        -H "Content-Type: application/json" \
        -d '{
          "userName": "johndoe",
          "firstName": "John",
          "lastName": "Doe",
          "passwordHash": "SecurePassword123!",
          "email": "john@example.com",
          "phoneNumber": "+1234567890"
        }'
      ```
      
      <Accordion title="Example Response">
        ```
        HTTP/1.1 201 Created
        Location: /api/users/1
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
